Cette commande permet de faire le commit, push et tag sur git.

# General
- pas d'emoji dans les messages de commit ou de tag

# Commit
- phrase très concise sur les changements, one liner suffit
- jamais de mention de claude code

# Push
- push sur la branche en cours

# Tag
- explication très concise, en anglais du nouveau tag
- tag en semver
- doit utiliser surtout des patchs
